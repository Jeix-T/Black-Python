[![License](https://img.shields.io/badge/license-MIT-blue.svg)](LICENSE)
[![Python](https://img.shields.io/badge/python-3.8+-green.svg)](https://www.python.org/)

Curso práctico de 10 semanas para desarrollar herramientas de hacking ético desde cero.

## 🚀 Inicio Rápido
1. Clona el repositorio: `git clone https://github.com/usuario/python-pentesting-course.git`
2. Lee el [DISCLAIMER.md](DISCLAIMER.md) (Obligatorio).
3. Configura tu laboratorio siguiendo la guía en [`00_laboratorio/`](00_laboratorio/).
4. Instala dependencias: `pip install -r REQUIREMENTS.txt`
